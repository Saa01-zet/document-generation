# Document Generation

Консольный скрипт для генерации документов (счёт и чек) с использованием паттернов Singleton и Factory.

## Описание

Проект демонстрирует реализацию двух паттернов проектирования:

- **Singleton** — класс `DocumentLogger` обеспечивает единый центр логирования
- **Factory** — функция `create_document()` создаёт документы нужного типа
